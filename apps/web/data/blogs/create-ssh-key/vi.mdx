## 1. Kiểm tra SSH Key hiện có

Bạn nên kiểm tra xem có SSH Key nào đã tồn tại trên MacOS của mình hay chưa.

Mở Terminal và chạy lệnh sau:

```bash
cd ~/.ssh
```

* Nếu bạn thấy thông báo "No such file or directory", điều đó có nghĩa là chưa có key nào tồn tại. [Chuyển đến bước 3](#create-a-new-key).
* Kiểm tra xem bạn đã có key hay chưa:

```bash
ls id_*
```

* Nếu đã có các key hiện có, bạn có thể mở các file này để lấy SSH Key và sử dụng ở bất cứ đâu bạn muốn (GitHub, GitLab, BitBucket,...)

## 2. Sao lưu SSH Key cũ

Trong trường hợp bạn đã có key nhưng muốn tạo một key mới thay vì sử dụng key cũ, bạn nên sao lưu chúng.

Quay lại Terminal của bạn, tại thư mục `.ssh`:

```bash
mkdir key_backups
cp id_* key_backups
```

## 3. Tạo SSH Key mới

### 3.1. Mở Terminal và nhập lệnh sau:

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```

**Lưu ý**: *Nếu bạn đang sử dụng hệ thống cũ không hỗ trợ thuật toán ED25519, hãy sử dụng lệnh dưới đây*:

```bash
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

Việc liên kết key với địa chỉ email của bạn sẽ giúp bạn dễ dàng nhận diện key đó sau này.

Bạn sẽ thấy kết quả tương tự như sau:

```bash
trung.nguyen@trungtmnguyen ~ % ssh-keygen -t ed25519 -C trungntm.97@gmail.com
Generating public/private ed25519 key pair.
Enter file in which to save the key (/Users/trung.nguyen/.ssh/id_ed25519):
```

### 3.2. Chỉ cần nhấn `Enter` để chấp nhận vị trí lưu và tên file mặc định. Nếu thư mục `.ssh` chưa tồn tại, hệ thống sẽ tự động tạo thư mục đó cho bạn.

### 3.3. Nhập và xác nhận lại passphrase khi được yêu cầu.
Toàn bộ quá trình sẽ tương tự như sau:

```bash
trung.nguyen@trungtmnguyen ~ % ssh-keygen -t ed25519 -C fperez@email.com
Generating public/private ed25519 key pair.
Enter file in which to save the key (/Users/trung.nguyen/.ssh/id_ed25519):
Enter passphrase (empty for no passphrase):
Enter same passphrase again:
Your identification has been saved in /Users/trung.nguyen/.ssh/id_ed25519.
Your public key has been saved in /Users/trung.nguyen/.ssh/id_ed25519.pub.
The key fingerprint is:
SHA256:gTVWKbn41z6JgBNu3wYjLC4abcdefghijklmnopqrstuvwxy trungntm.97@gmail.com
The key's randomart image is:
+--[ED25519 256]--+
|==+.    +o..     |
|.oE.   +o..      |
|    . ...o       |
|     .o...       |
|     oo+S  .     |
|  + ..B = . .    |
|.+.+.oo+ * o .   |
|o++.o+  . + +    |
|B+ o.    .   .   |
+----[SHA256]-----+
trung.nguyen@trungtmnguyen ~ %
```

### 3.4. Well done. Bạn có thể sử dụng SSH Key mới để cấu hình cho hệ thống Git hoặc máy chủ từ xa.
